二叉树遍历

2020-04-13 约 721 字 预计阅读 2 分钟

二叉树结构

 public class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode(int x) {
            val = x;
        }
    }

前序遍历

  1. 把二叉树的根节点放进栈。
  2. 如果栈不为空，从栈中取一个节点，把该节点放入容器的尾部；如果该节点的右子树不为空，则把右子树放入栈；如果该节点的左子树不为空，则把左子树放入栈中。
  3. 一直重复步骤2，直到栈为空。
/**
     * 前序遍历 根-左-右
     *
     * @param root
     * @return
     */
    public static List<Integer> firstOrderTraversal(TreeNode root) {

        List<Integer> list = new ArrayList<>();
        if(root==null){
            return list;
        }
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        stack.push(cur);
        while (!stack.isEmpty()) {
            cur = stack.pop();
            list.add(cur.val);
            if (cur.right!= null) {
                stack.push(cur.right);
            }
            if (cur.left!= null) {
                stack.push(cur.left);
            }
        }
        return list;
    }

中序遍历

  1. 进入while循环，接着把根节点及其所有左子节点放入栈中
  2. 从栈中取出一个节点，把该节点放入容器的尾部；如果该节点的右子节点不为空，则把右子节点及其右子节点的所有子节点放入队列。
  3. 一直重复2知道栈为空并且当前节点也为空则退出循环。
/**
     * 中序遍历 左-根-右
     *    
   
     * @param root
     * @return
     */
    public static List<Integer> inorderTraversal(TreeNode root) {

        List<Integer> list = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        while (cur != null || !stack.isEmpty()) {
            if (cur != null) {
                stack.push(cur);//1-2-3
                cur = cur.left; //null-3-null
            } else {
                cur = stack.pop();//1-3-2
                list.add(cur.val);//1-3-2
                cur = cur.right;//2-null-null
            }
        }
        return list;
    }

后序遍历

  1. 把根节点放入栈。
  2. 如果栈不为空，从栈中取一个节点，把该节点放入容器的尾部；如果该节点的左子树不为空，则把左子树放入栈；如果该节点的右子树不为空，则把右子树放入栈中。（注意与前序遍历的区别）
  3. 一直重复步骤2，直到栈为空。
/**
     * 后序遍历 左-右-根
     *
     * @param root
     * @return
     */
    public static List<Integer> lastOrderTraversal(TreeNode root) {

        List<Integer> list = new ArrayList<>();
        if(root == null){
            return list;
        } 
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        stack.push(cur);
        while (!stack.isEmpty()) {
            cur = stack.pop();
            if (cur.left != null) {
                stack.push(cur.left);
            }
            if (cur.right != null) {
                stack.push(cur.right);
            }
            list.add(0, cur.val);
        }
        return list;
    }

层序遍历-正序

  /** 层序遍历
     * [
     * [3],
     * [9,20],
     * [15,7]
     * ]
     *
     * //输出
     * [[3], [9, 20], [7, 15]]
     * @param root
     * @return
     */
    public static List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        Queue<TreeNode> queue = new LinkedList<>();
        queue.add(root);
        while (!queue.isEmpty()) {
            int level = queue.size();
            List<Integer> list = new ArrayList<>();
            while (level > 0) {
                TreeNode node = queue.poll();
                list.add(node.val);
                if (node.left != null) {
                    queue.add(node.left);
                }
                if (node.right != null) {
                    queue.add(node.right);
                }
                level--;
            }
            res.add(list);
        }
        return res;
    }

层序遍历-倒序

 /**层序遍历
     * [
     * [3],
     * [9,20],
     * [15,7]
     * ]
     *
     * //输出
     * [[7, 15], [9, 20], [3]]
     *
     * @param root
     * @return
     */
    public static List<List<Integer>> levelOrder2(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        Queue<TreeNode> queue = new LinkedList<>();
        queue.add(root);
        while (!queue.isEmpty()) {
            int level = queue.size();
            List<Integer> list = new ArrayList<>();
            while (level > 0) {
                TreeNode node = queue.poll();
                list.add(node.val);
                if (node.left != null) {
                    queue.add(node.left);
                }
                if (node.right != null) {
                    queue.add(node.right);
                }
                level--;
            }
            res.add(0,list);
        }
        return res;
    }
